# Global Socio-Economic Indicator Dashboard

## 📌 Overview
This project analyzes global socio-economic indicators across countries, regions, and income groups to identify patterns, disparities, and relationships between economic factors and indicator values. The analysis is performed using Python and visualized through an interactive dashboard.

---

## 🎯 Problem Statement
The objective of this project is to study how socio-economic indicators vary across geographical regions and income groups, and to analyze how economic factors such as GDP per capita influence these indicators.

---

## 🎯 Objectives
- Perform data cleaning and preprocessing on a real-world dataset  
- Conduct exploratory data analysis (EDA)  
- Analyze distribution and trends of indicator values  
- Compare indicators across regions and income groups  
- Study the relationship between GDP and socio-economic indicators  
- Build an interactive dashboard for data visualization  

---

## 🛠 Tools & Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- hvPlot  
- Panel  

---

## 📊 Dataset Description
The dataset contains global socio-economic indicators collected across multiple countries and years. It includes features such as:

- Country  
- Region  
- Year  
- Income Group  
- GDP per Capita  
- Indicator Values  
- Sample Size  

The dataset required cleaning and preprocessing due to missing values and inconsistencies.

---

## ⚙️ Data Preprocessing
- Removed irrelevant and highly missing columns  
- Handled missing values using mean, median, and mode  
- Converted data into suitable formats  
- Created new features such as:
  - GDP Category  
  - Indicator Value Category  

---

## 📈 Exploratory Data Analysis
The following analyses were performed:

- Distribution analysis using histograms  
- Outlier detection using boxplots  
- Region-wise comparison of indicator values  
- Income group-based analysis  
- Summary statistics and data insights  

---

## 📊 Statistical Analysis
- Descriptive statistics (mean, median, standard deviation)  
- Correlation analysis between numerical variables  
- Skewness and kurtosis analysis  
- Group-wise comparison across regions and income groups  

---

## 📉 Key Visualizations
- Top 10 Countries by Indicator Value  
- Distribution of Indicator Values  
- GDP vs Indicator Scatter Plot (with Income Group legend)  
- Regional Distribution Pie Chart  
- Correlation Heatmap  

---

## 📊 Dashboard
An interactive dashboard was developed using Panel, featuring:

- Year-based filtering using buttons  
- Dynamic and interactive visualizations  
- Clean and structured layout  
- Comparative and analytical insights
